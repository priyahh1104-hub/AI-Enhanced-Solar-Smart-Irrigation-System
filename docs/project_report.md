# AI-Enhanced Solar Smart Irrigation System

## Objective
To develop an intelligent irrigation system that conserves water using AI algorithms and weather prediction.

## Components Used
- ESP32 Development Board
- Soil Moisture Sensor
- DHT11 Sensor
- Relay Module
- Water Pump
- Solar Panel

## Features
- Soil moisture monitoring
- Automatic pump control
- Weather prediction
- Mobile app monitoring
- Solar-powered operation

## Applications
- Smart agriculture
- Water conservation
- Precision farming
